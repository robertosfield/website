Detailed Description

ViewDescriptorSetLayout is a proxy class that uses the ViewDependentState's descriptorSetLayout as the DescriptorSetLayout to use. Used in pipelines that wish to utilize lights and other view dependent data provided by the View::viewDependentState. Use in combination with the BindViewDescriptorSet.
